# Diet Ledger (食飯帳本)
IOS APP for Diet Ledger.
## 設計目的：
一款專為飲食消費記錄所設計出的APP，透過本APP，您可以獲得以下的好處，(1)了解自己的飲食狀況，(2)了解自己的飲食喜好與習慣，(3)了解自己的飲食地圖，(4)掌握自身的飲食開銷，(5)掌握自身的飲食紀錄，(5)掌握自身的飲食健康。
## 技術內容：
本作品運用資料庫，GPS定位資訊與手機APP的便利性建構出方便又輕巧的飲食記帳本。
